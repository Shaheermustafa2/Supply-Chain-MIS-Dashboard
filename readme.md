### Supply Chain MIS Dashboard (Excel)



##### Developed an interactive Supply Chain MIS Dashboard in Microsoft Excel to monitor key operational metrics such as revenue, order volume, supplier performance, inventory status, and delivery efficiency. The dashboard leverages Pivot Tables, Pivot Charts, Slicers, and KPI reporting to transform raw transactional data into actionable business insights.



### Key Features

* ##### Interactive dashboard with slicers for Region, Supplier, Product Category, and Delivery Status.
* ##### KPI tracking including Total Revenue, Total Orders, On-Time Delivery %, Reorder Items, and Total Profit.
* ##### Visual analysis of monthly revenue trends, regional performance, supplier efficiency, inventory alerts, and profit distribution.
* ##### Built using 1,000+ supply chain transaction records to simulate real-world MIS reporting.



### Tools \& Technologies

* ##### Microsoft Excel
* ##### Pivot Tables \& Pivot Charts
* ##### Slicers
* ##### Conditional Formatting
* ##### Business KPI Reporting
